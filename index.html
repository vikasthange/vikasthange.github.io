About me

I'm Project Manager (Testing) and Automation Test Architect from Toronto, Canada, with 13+ years of experience in IT Software development and testing. I enjoy setting up automation testing workflow by design, develop and implementing various automation testing frameworks & standrds.

My job is to manage the automation testing projects which covers design, develop & implement automation testing frameworks for functional and non functional testing requirements. Moreover, onboarding manual testing teams to automation workflow by enterprise level common automation test frameworks.

Alwyas my aim has been to replace all manual testing with automated testing workflow across projects to increase reliblity on automation test execution with short regression test cycle. In my past experience I have sucessfully created and implemented various frameworks based on DataDriven, Keyword Driven, Page Object Model, Hybrid, BDD.

I'm also a part time blogger, Youtubber and former corporate trainer on testing technology. I enjoy discussion on taking automation testing to next level, latest updates / tools in automation industary.
